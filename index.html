Mombasa Python User Group

Currently there isn't a python user group in Mombasa, Kenya. Britone Mwasaru is starting a technology community in Technical University of Mombasa. For the students that are planning to venture into programming will be introduced to python.

Why a python user group?

Python is a language that is easy to learn therefore a good introductory language especially to the first year students. This is show how fun and easy programming can be hence get rid of the fear of programming.

Cheers to a journey of self discovery :)
