Mombasa Python User Group

Currently there isn't a python user group in Mombasa, Kenya. Britone Mwasaru is starting a technology community in Technical University of Mombasa. For the students that are planning to venture into programming will be introduced to python. Planned launch date on Friday 8th November at 2p.m EAT.

Why a python user group?

Python is a language that is easy to learn therefore a good introductory language especially to the first year students. This is to show how fun and easy programming can be hence get rid of the fear of programming.

Cheers to a journey of self discovery :)

****8th November 2013 Launch Day

Despite the timings problems, we posted the posters 4 hours to the event, the attendance was surprisingly good.

  • Started with the code.org video on learning how to code so as to create the sense that programming is not difficult.
  • We explained why we were using python and examples of companies using python. We did installation of python on windows (a large number had windows Operating System).
  • It was a hand ons session, they followed along with me. In case of any problem they had the freedom to shout and interrupt me.
  • We discussed how to use the python terminal and the IDLE. Explained to reasons to find a better editor for python.
  • Britone did a demo of web application that he is working on. He explained the Django python web framework.

The attendees exclaimed that they previously thought that programming was hard but after the session believed that programming is not that hard. One gentleman asked about how to use python to build control systems. It was explained to him that there are open source libraries such as SciPy and NumPy that would be of use to him. Another asked how he can use python for networking, he was answered that there are libraries such as Twisted that would be of help to him.

Finally, we discussed about the Mombasa Technology Community and how they can get involved. The fellow students were encouraged to collaborate and learn from each other.
